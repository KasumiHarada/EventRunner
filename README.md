# EventRunner

## サイトURL
最後にURLをここに貼る

## 概要
イベントを主催、参加できるWebアプリを開発する。  
誰もが簡単に、イベントの主催及び参加表明ができるように設計する。  
特定のコミュニティでの利用を想定している（不特定多数でない）。  
納期は2週間とする（2020/7/20〜2020/8/3）。
## 使用方法

ユーザーは会員登録して、Webサイトへログインする。　
イベントを探したい場合は、「イベントを探す」より気になるイベントを探す。

イベントを登録し、他の参加者を募ることができる
